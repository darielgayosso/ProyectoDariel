# Proyecto LMS - Plataforma de Gestión de Aprendizaje

Este proyecto es un **Sistema de Gestión de Aprendizaje (LMS)** profesional desarrollado con React y Supabase. Ofrece una solución completa para la gestión educativa con roles diferenciados para Administradores, Docentes y Estudiantes.

## 🚀 Características Principales

*   **Autenticación Segura**: Sistema de login y registro con Supabase y soporte para Google OAuth.
*   **Gestión por Roles**: Paneles personalizados para cada tipo de usuario.
*   **Estructura de Cursos**: Organización por unidades, materiales multimedia y tareas.
*   **Evaluaciones Integradas**: Creador de cuestionarios avanzado y resolución con autocalificación.
*   **Muro de Clase**: Sistema de comunicación bidireccional entre docentes y alumnos.
*   **Diseño Premium**: Interfaz moderna, responsive y con soporte nativo para **Modo Oscuro**.

## 🛠️ Tecnologías

*   **Frontend**: React 19, Tailwind CSS, Lucide Icons.
*   **Backend**: Supabase (PostgreSQL, Auth, Realtime).
*   **Routing**: React Router DOM v7.

## ⚙️ Configuración Rápida

1. Instala las dependencias: `npm install`
2. Configura tu `.env` con las credenciales de Supabase.
3. Inicia el proyecto: `npm start`
